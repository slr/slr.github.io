чтобы играть в UT2004 сегодня, достаточно двух простых правок конфига.

today to play UT2004 two easy edits of configs are enough.

во-первых, чтобы иметь возможность подключаться к мастер-серверу OpenSpy, необходимо следующие строки в файле System/UT2004.ini

first, to be able to connect to OpenSpy master server one needs to replace the following strings in file System/UT2004.ini

MasterServerList=(Address="ut2004master1.epicgames.com",Port=28902)

MasterServerList=(Address="ut2004master2.epicgames.com",Port=28902)

заменить на

with

MasterServerList=(Address="utmaster.openspy.net",Port=28902)

во-вторых, чтобы комфортно играть на широкоформатном мониторе, нужно распаковать в папку с игрой архив с патчем из репозитория alexstrout/foxWSFix-UT2k4, а затем следующие строки (все из них) в файле System\User.ini

second, to play with ease on a widescreen display one needs to extract to game directory an archive with a patch from alexstrout/foxWSFix-UT2k4 repository and then replace the following strings (all of them) in file System\User.ini

InputClass=Class'Engine.PlayerInput'

заменить на

with

InputClass=Class'foxWSFix.foxPlayerInput'

разрешение можно поменять через встроенную игровую консоль (нажми ` на клавиатуре)

resolution can be adjusted via the game's built-in console command (press ` on a keyboard)

SetRes 1920x1080
